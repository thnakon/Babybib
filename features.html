Babybib

ระบบจัดการบรรณานุกรมอัจฉริยะแบบครบวงจร (Smart Bibliography Management System) ออกแบบมาเพื่อนักวิจัยและนักศึกษาโดยเฉพาะ

สถาปัตยกรรมระบบ

Back-end Architecture

ขับเคลื่อนด้วย PHP 8.2 ที่มีประสิทธิภาพสูง รองรับการทำงานแบบ Dynamic และการจัดการ Session ที่ปลอดภัย

  • โครงสร้างแบบ Modular Clean Code
  • เชื่อมต่อฐานข้อมูลผ่านระบบ PDO
  • API-First Design ประมวลผลรวดเร็ว

Data Management

ใช้ฐานข้อมูล MySQL ในการจัดเก็บข้อมูลสมาชิก โครงการ และประวัติการทำงานอย่างเป็นระบบ

  • Relational Database Design
  • ระบบจัดเก็บ Cache คลังข้อมูลอัจฉริยะ
  • รองรับข้อมูลบรรณานุกรม 10+ รูปแบบ

Front-end Design

ออกแบบส่วนติดต่อผู้ใช้ด้วยเทคโนโลยีชั้นนำ เน้นความสวยงาม ทันสมัย และใช้งานง่าย (User-Centric)

  • Vanilla CSS 3 (Variable-driven)
  • Responsive Mobile-First Layout
  • Glassmorphism Premium UI

คุณสมบัติและฟีเจอร์เด่น

Smart Search v3.1

ระบบค้นหาอัจฉริยะที่สามารถตรวจจับประเภทข้อมูลได้โดยอัตโนมัติ (Auto-Detection) ไม่ว่าจะเป็น ISBN, DOI, URL หรือคีย์เวิร์ดภาษาไทย/อังกฤษ ระบบจะเลือกดึงข้อมูลจากแหล่งที่น่าเชื่อถือที่สุดมาให้ทันที

Multi-Format Export

ส่งออกข้อมูลบรรณานุกรมไปยังไฟล์มาตรฐาน เพื่อนำไปใช้งานต่อในเอกสารวิชาการได้ทันที

  • Microsoft Word (.DOCX) - ฟอนต์ไทยที่คุ้นเคย
  • PDF Preview & Print
  • Copy to Clipboard (One Click)

Project Workspace

จัดการบรรณานุกรมแยกตามโครงการวิจัย ช่วยให้การทำรายงานหลายฉบับพร้อมกันเป็นเรื่องง่าย

  • แยกคลังข้อมูลตามหมวดหมู่
  • ระบบจัดเรียงลำดับ ก-ฮ / A-Z อัตโนมัติ
  • ย้ายข้อมูลระหว่างโครงการได้อิสระ

LIS Student Features

ระบบพิเศษสำหรับนักศึกษาสารสนเทศศาสตร์ที่ต้องการการจัดการเชิงลึก

  • ระบบสถานะนักศึกษา LIS
  • ประวัติการทำงาน (Work History)
  • การจัดการโปรไฟล์แบบละเอียด

ความปลอดภัยและการตั้งค่า

System Security

ปกป้องข้อมูลของผู้ใช้ด้วยมาตรการรักษาความปลอดภัยระดับมาตรฐานสากล

  • Rate Limiting ป้องกันการยิง API
  • CSRF Protection ป้องกันการโจมตีข้ามไซต์
  • Input Sanitization ป้องกัน SQL Injection
  • Session Secure Hijacking Prevention

Admin Control Center

ระบบหลังบ้านที่ทรงพลังสำหรับผู้ดูแลระบบในการตรวจสอบและจัดการภาพรวม

  • ระบบจัดการสมาชิกและสิทธิ์การใช้งาน
  • Activity Logging ตรวจสอบย้อนหลังได้ทุกจังหวะ
  • Dashboard สรุปสถิติการใช้งานแบบ Real-time
  • ระบบจัดการการตั้งค่าระบบส่วนกลาง
